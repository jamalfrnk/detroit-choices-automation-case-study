## Screenshot Guidelines

- Crop screenshots to the relevant content — avoid excess browser chrome
- Redact any personal customer data, credentials, or private endpoints before committing
- Use PNG for static screenshots, GIF for short screen recordings
- Name files descriptively using lowercase and hyphens (e.g., `zapier-order-intake-zap.png`)
